# &lt;pre&gt;

### 说明
pre 标签可定义预格式化的文本。
被包围在 pre 标签 元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体。


```html
<pre>
Text in a pre element
is displayed in a fixed-width
font, and it preserves
both      spaces and
line breaks
</pre>
```
