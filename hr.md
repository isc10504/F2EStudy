# &lt;hr&gt;

### 说明
&lt;hr&gt; 标签定义 HTML 页面中的主题变化（比如话题的转移），并显示为一条水平线。
&lt;hr&gt; 元素被用来分隔 HTML 页面中的内容（或者定义一个变化）。


```html
<h1>HTML</h1>
<p>HTML is a language for describing web pages.....</p>

<hr>

<h1>CSS</h1>
<p>CSS defines how to display HTML elements.....</p>
```
